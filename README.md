jamgame
=======

Game Jam Graz Sep 2014 project



Minimalvariante:
- 1 Level, fixe Elemente (Brot, Milch etc.)
- Controls: links/rechts steuern
- Bewegungsgeschwindigkeit fix und nicht vom Spieler veränderbar
- Kollision zwischen den Spielern (nur Richtungsablenkung)
- Win Condition: Wenn ein Spieler vom Tisch fällt, hat er verloren
- Früchte sammeln
- Spur legen vom Spieler gesteuert (nur solange der Fruchtvorrat reicht)
- Wenn man auf die Spur kommt, dann wird man schneller
- Sounds:
 - Runterfallen
 - Marmeladen Squish
 - Gewonnen (Fanfare)
- Start Screen
- End Screen
- Credits

Stretch Goals:
- unterschiedliche Früchte mit unterschiedlichen Kräften
- Elemente am Tisch randomisieren
- Jam Fight (Gläser können zerstört werden)
